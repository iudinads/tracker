# Трекер

Личное приложение для дел и тренировок.

## Запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000)

## Доступ с iPhone / Mac в локальной сети

```bash
npm run dev -- -H 0.0.0.0
```

Затем откройте `http://<IP вашего Mac>:3000` на iPhone.

## Возможности

- **Дела** — кастомные разделы, списки задач, даты, статусы
- **Тренировки** — разделы, сессии с упражнениями и подходами, сравнение прогресса

Данные хранятся локально в `data/tracker.json`.
